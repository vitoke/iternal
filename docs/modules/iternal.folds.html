histogram

  • histogram<E>(sortBy?: "TOP" | "BOTTOM", amount?: undefined | number): Folder<E, Histogram<E>>
    • Defined in iternal-fold/folds/index.ts:407

    Returns a folder that creates a histogram of the received elements. That is, it creates a Map with the elements as keys and the amount of occurrances as values.

    example
    Fold.fold('adcbcd', Folds.histogram())
result: Map('a' -> 1, 'd' -> 2, 'c' -> 2, 'b' -> 1)

    Type parameters

    • E

      the element type

    Parameters

    • Optional sortBy: "TOP" | "BOTTOM"

      if undefined will return the histogram in value order, if value is 'TOP' it will order the histogram from highest to lowest, otherwise it will order the histogram from lowest to highest frequency.

    • Optional amount: undefined | number

      if sortBy is specified, this parameter limits the amount of results

    Returns Folder<E, Histogram<E>>
